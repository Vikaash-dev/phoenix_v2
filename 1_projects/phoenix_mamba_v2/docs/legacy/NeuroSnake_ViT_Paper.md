# NeuroSnake-ViT: A Hybrid Architecture for Brain Tumor Detection

## Abstract

This paper introduces NeuroSnake-ViT, a novel hybrid deep learning architecture for brain tumor detection. NeuroSnake-ViT combines the geometric adaptability of Dynamic Snake Convolutions with the global context awareness of Vision Transformers to achieve state-of-the-art performance on the BraTS 2020 dataset.
